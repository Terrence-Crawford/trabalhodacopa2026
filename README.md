# Modelo de Previsão da Copa

Este projeto utiliza um modelo de regressão de Poisson para prever resultados de partidas de futebol a partir de estatísticas das equipes.

## Arquivos

- `modelo_copa.ipynb` – Notebook contendo todo o desenvolvimento e execução do modelo.
- `teams_match_features.csv` – Base de dados utilizada pelo notebook.

## Requisitos

É necessário ter o Python 3 instalado e as seguintes bibliotecas:

```bash
pip install pandas numpy matplotlib scikit-learn tqdm juypyter notebook
```

## Como executar

1. Clone ou baixe este repositório.
2. Instale as dependências acima.
3. Certifique-se de que o arquivo `teams_match_features.csv` esteja na mesma pasta do notebook.
4. Abra o arquivo `modelo_copa.ipynb` no Jupyter Notebook ou VS Code e execute as células em ordem.
